# Bulk File Renamer

A powerful and user-friendly bulk file renaming application built with Python and Tkinter.

## Features

- **Find and Replace**: Search and replace text in filenames
- **Prefix/Suffix**: Add prefixes or suffixes to filenames
- **Case Conversion**: Convert filenames to lowercase, uppercase, or title case
- **Numbering**: Add sequential numbers as prefix, suffix, or replace entire name
- **Extension Management**: Change file extensions to lowercase or uppercase
- **Recursive Processing**: Include files in subdirectories
- **Preview**: See changes before applying them
- **Conflict Detection**: Warns about filename conflicts
- **Undo Support**: Can't undo directly, but shows preview before execution

## Requirements

- Python 3.6 or higher
- No external packages required (uses standard library only)

## Installation

1. Download the `file_renamer.py` file
2. Save it to your preferred location
3. Open a terminal/command prompt in that location

## Usage

### Running the Application
